## 编程语言

### hello world

#### C - 1972

```c
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}
```

#### C++ - 1980

```c
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}
```

#### Python - 1991

```python
print("Hello, World!")
```

#### Java - 1995

```java
class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
```



#### Go - 2009

```go
package main

import "fmt"

func main() {
  fmt.Println("Hello, World!")
}
```

#### Rust - 2010

```rust
fn main() {
    println("Hello, World!");
}
```
